# Corvalan Tech GitHub Pages Website

This is a static website ready for GitHub Pages.

## Files

- `index.html` — website content
- `style.css` — website design
- `script.js` — mobile menu and current year

## Before publishing

Edit these placeholders in `index.html`:

1. `info@corvalantech.com`
2. `Replace with phone number`
3. `Texas and surrounding regions`
4. The About section paragraph
5. Any service wording you want changed

## Publish on GitHub Pages

1. Create a GitHub account if you do not have one.
2. Create a new public repository named `corvalantech-site`.
3. Upload these files into the repository root.
4. Go to Settings → Pages.
5. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. GitHub will give you a free `github.io` website link.

## Use your own domain

After the free site works, add your domain in Settings → Pages → Custom domain.
You will also need to update DNS records where your domain is registered.
