## Upload to GitHub

1. Open your CorvalanTech repository.
2. Delete the old files if needed.
3. Upload all files from this folder, not the ZIP file.
4. Commit changes.
5. Go to Settings > Pages.
6. Use Deploy from a branch, main, /root.

## Notes

The App Store buttons are placeholders. Replace `#app-store` in the app pages with the real App Store links when available.
The LinkedIn link is also a placeholder.
