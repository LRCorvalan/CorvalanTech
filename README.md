# CorvalanTech GitHub Pages Site

Upload these files to the root of your GitHub repository. Do not upload the ZIP file itself.

Required files:
- index.html
- style.css
- daysnap.html
- phopan.html
- cuetrack.html
- privacy-daysnap.html
- privacy-phopan.html
- privacy-cuetrack.html

Then go to Settings > Pages and publish from the main branch, root folder.

- Phopan: https://apps.apple.com/us/app/phopan/id6670329857
